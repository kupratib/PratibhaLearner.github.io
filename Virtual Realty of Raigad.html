Virtual Reality Raigad Fort System

This project is 3D view of Raigad Fort that can be explored interactively at a personal computer, usually by manipulating keys or the mouse so that the content of the image moves in some direction or zooms in or out. More sophisticated efforts involve such approaches as wrap-around display screens, actual rooms augmented with wearable computers, and haptics devices that let you feel the display images.

his system is used to explore the Raigad Fort and it reduce the time, money etc. This system is just like game but in game we can play and win. In this system we know more things of each place in Raigad Fort by rendering in this system.

The structure of the Raigad fort in reality and in our system is approximately same so people can easily recognize it. Many game engines are present to made this system but we use Unity 3D because it is easy to learn and implement. For scripting we used C Sharp language, for making pillars, statues etc. 3D images we used 3D paint.
